# 黑白模糊老照片 → 彩色高清照片完整教程

## 🎯 教程目标

将一张**黑白、模糊、有噪点**的老照片，修复成**彩色、清晰、高清**的照片。

**修复效果**：
- ✅ 黑白 → 彩色（AI智能上色）
- ✅ 模糊 → 清晰（多重锐化）
- ✅ 低质 → 高清（提升分辨率和细节）
- ✅ 噪点 → 干净（智能降噪）
- ✅ 损伤 → 完好（修复划痕折痕）

---

## 📋 准备工作

### 所需工具
- Adobe Photoshop CS6 或更高版本
- `老照片修复.jsx` 脚本
- （可选）AI增强工具如 Topaz Gigapixel AI

### 照片要求
| 要求 | 说明 |
|------|------|
| 格式 | JPG、PNG、TIFF 均可 |
| 分辨率 | 最低 500×500，推荐 1000×1000 以上 |
| 内容 | 人物肖像或风景照均可 |
| 损伤程度 | 轻微到中度损伤最佳 |

---

## 🚀 完整修复流程（分步指南）

### 阶段一：高清化处理（可选但推荐）

**目的**：提升照片分辨率，为后续修复提供更多细节

#### 方法1：使用Photoshop内置功能

1. **打开照片**
   ```
   文件 → 打开 → 选择黑白老照片
   ```

2. **提升分辨率**
   ```
   图像 → 图像大小
   - 勾选"重新采样"
   - 选择"保留细节 2.0（放大）"
   - 分辨率改为：300 DPI
   - 宽度/高度提升 2-3 倍（如 1000px → 2000px）
   - 点击确定
   ```

3. **初次锐化**
   ```
   滤镜 → 锐化 → USM锐化
   - 数量：80%
   - 半径：1.0
   - 阈值：3
   ```

#### 方法2：使用AI增强工具（推荐）

如果你有 **Topaz Gigapixel AI** 或 **Waifu2x** 等工具：

1. 使用AI工具将照片放大 2-4 倍
2. 选择"照片修复"模式
3. 导出放大后的照片到Photoshop

**优势**：AI能更好地重建细节和纹理

---

### 阶段二：基础修复（去模糊、去噪点）

**目的**：修复模糊和噪点问题

1. **运行修复脚本**
   ```
   文件 → 脚本 → 浏览 → 选择"老照片修复.jsx"
   ```

2. **配置修复选项**
   - **处理强度**：选择"**重度**"（因为是模糊照片）
   - ✅ 自动调整色阶/对比度
   - ✅ 色彩平衡修正（即使黑白照片也要勾选）
   - ✅ 减少噪点/划痕（**重要**）
   - ✅ 智能锐化（**重要**）
   - **色调处理**：选择"**转换为黑白**"（确保纯净黑白）

3. **点击"开始修复"**

4. **脚本会创建以下图层**：
   ```
   📄 色阶调整
   📄 色彩平衡
   📄 降噪层
   📄 锐化层
   📄 整体提亮
   📄 黑白效果
   ```

5. **合并可见图层**（为上色做准备）
   ```
   Ctrl+Shift+Alt+E (Windows)
   Cmd+Shift+Option+E (Mac)
   ```

---

### 阶段三：手动去模糊（关键步骤）

**目的**：进一步增强清晰度

#### 1. 高反差保留锐化

```
1. 复制合并后的图层 (Ctrl+J / Cmd+J)
2. 滤镜 → 其他 → 高反差保留
   - 半径：2.0-4.0（根据模糊程度调整）
3. 设置混合模式为"叠加"或"柔光"
4. 调整不透明度：50-80%
```

#### 2. 智能锐化（二次）

```
1. 滤镜 → 锐化 → 智能锐化
2. 基本设置：
   - 数量：100-150%
   - 半径：1.5-2.0
   - 减少"杂色"：选中
```

#### 3. 局部锐化（人物面部）

```
1. 选择"模糊工具"或"锐化工具"
2. 选择面部区域，用锐化工具加强细节
3. 用模糊工具处理背景，突出主体
```

---

### 阶段四：AI智能上色

**目的**：为黑白照片添加真实色彩

#### 1. 运行脚本上色功能

```
文件 → 脚本 → 浏览 → "老照片修复.jsx"
```

#### 2. 配置上色选项

根据照片类型选择：

| 照片类型 | 上色方案 | 上色强度 |
|---------|---------|---------|
| 👤 人物肖像 | **人物肖像（肤色优先）** | 适中 |
| 👨‍👩‍👧‍👦 全家福 | **智能自动上色** | 轻微/适中 |
| 🌳 风景照 | **风景照片（自然绿蓝）** | 适中 |
| 🏠 建筑街景 | **智能自动上色** | 适中 |
| 🎨 艺术创作 | **复古暖色调** | 强烈 |

#### 3. 修复选项设置

- **处理强度**：轻度（避免过度处理彩色照片）
- ✅ 只勾选"自动调整色阶/对比度"
- ❌ 其他选项不勾选（避免重复处理）
- ✅ **启用智能上色**

#### 4. 点击"开始修复"

脚本会创建 **🎨 上色效果组**，包含多个颜色层。

---

### 阶段五：精细调整上色效果

**目的**：让上色效果更加自然真实

#### 1. 调整颜色层不透明度

打开"🎨 上色效果组"，逐个调整：

```
🎨 基础暖色层    → 不透明度：20-30%
🎨 天空蓝调      → 不透明度：15-25%
🎨 植被绿调      → 不透明度：15-20%
🎨 肤色暖调      → 不透明度：20-30%
```

**技巧**：
- 人物为主的照片：提高"肤色暖调"，降低其他颜色
- 风景为主：提高"天空蓝"和"植被绿"
- 不满意的颜色层可以直接删除

#### 2. 局部上色（重要！）

真实照片不是全局均匀上色的，需要局部调整：

```
1. 选择某个颜色层（如"肤色暖调"）
2. 添加图层蒙版
3. 使用软画笔工具（B）
   - 黑色画笔：涂抹不需要上色的区域（隐藏颜色）
   - 白色画笔：涂抹需要上色的区域（显示颜色）
   - 灰色画笔：半透明效果
4. 画笔不透明度：30-50%
```

**实例**：
- **人物脸部**：只在脸部显示"肤色暖调"
- **衣服**：降低或隐藏颜色层，保持原色或手动调色
- **天空**：只在天空区域显示"天空蓝"
- **草地**：只在植被区域显示"植被绿"

#### 3. 手动调整颜色

如果某些颜色不理想：

```
1. 双击颜色图层的缩略图
2. 在拾色器中调整颜色
3. 或新建"色相/饱和度"调整图层
4. 用蒙版限制作用范围
```

---

### 阶段六：最终优化

**目的**：提升整体质量，达到高清效果

#### 1. 色彩平衡微调

```
1. 新建"色彩平衡"调整图层
2. 调整：
   - 阴影：+5 青，-5 黄
   - 中间调：根据需要微调
   - 高光：+3 蓝
```

#### 2. 色阶/曲线优化

```
1. 新建"色阶"或"曲线"调整图层
2. 稍微提亮中间调
3. 增加对比度（S型曲线）
```

#### 3. 最终锐化

```
1. 合并可见图层 (Ctrl+Shift+Alt+E)
2. 滤镜 → 锐化 → USM锐化
   - 数量：60-80%
   - 半径：0.8-1.2
   - 阈值：2-4
```

#### 4. 去噪（如有必要）

```
1. 滤镜 → 杂色 → 减少杂色
   - 强度：5-8
   - 保留细节：60-80%
```

---

### 阶段七：输出保存

**目的**：保存高清彩色照片

#### 保存格式建议

| 用途 | 格式 | 设置 |
|------|------|------|
| 打印/存档 | **TIFF** | 无压缩，LZW压缩 |
| 网络/分享 | **JPEG** | 质量：10-12（最高） |
| 继续编辑 | **PSD** | 保留所有图层 |
| 通用 | **PNG** | 无损压缩 |

#### 保存步骤

```
1. 文件 → 存储为
2. 选择格式
3. 命名：原文件名_修复上色
4. 点击保存
```

## 🎨 实战案例

### 案例1：人物肖像照片

**原始状态**：500×500 px，黑白，模糊，脸部噪点

**修复流程**：
1. **高清化**：放大至 1500×1500 px（保留细节 2.0）
2. **基础修复**：运行脚本，重度强度
3. **手动锐化**：高反差保留（半径 3.0）+ 叠加模式
4. **上色**：选择"人物肖像"方案，适中强度
5. **精细调整**：
   - 肤色层不透明度：35%
   - 添加蒙版，只显示脸部和手部
   - 衣服保持黑白或单独上色
6. **最终锐化**：USM 70% / 1.0 / 3

**结果**：1500×1500 px 彩色高清照片

---

### 案例2：风景老照片

**原始状态**：800×600 px，黑白，模糊，天空过曝

**修复流程**：
1. **高清化**：AI放大至 2400×1800 px
2. **基础修复**：脚本重度处理
3. **上色**：选择"风景照片"方案
4. **精细调整**：
   - 天空蓝：30% 不透明度
   - 草地绿：25% 不透明度
   - 用蒙版分离天空和地面
5. **局部修复**：
   - 修复划痕（仿制图章）
   - 提高天空细节（色阶调整）

**结果**：2400×1800 px 彩色风景照

## ⚠️ 常见问题及解决方案

| 问题 | 原因 | 解决方法 |
|------|------|---------|
| **上色后颜色不自然** | 上色强度过高 | 降低各颜色层不透明度到 20% 以下 |
| **脸部肤色发绿** | 植被绿调影响脸部 | 在"植被绿"层添加蒙版，隐藏脸部区域 |
| **照片仍然模糊** | 锐化不足 | 重复"高反差保留"步骤，或增加锐化半径 |
| **噪点明显** | 降噪不够或过度锐化 | 添加"减少杂色"滤镜，降低锐化强度 |
| **颜色过饱和** | 多个颜色层叠加 | 降低部分颜色层不透明度，或删除多余层 |
| **某些区域上色错误** | 全局上色 | 使用图层蒙版，局部控制上色范围 |

## 💡 专业技巧

### 1. 分层上色法

对于复杂的照片，建议**分区域上色**：

```
第一次运行脚本：
  - 只对脸部区域上色（用蒙版限制）

第二次运行脚本：
  - 只对衣服上色

第三次运行脚本：
  - 只对背景上色
```

### 2. 混合模式技巧

不同的混合模式产生不同效果：

```
COLOR 模式：     只改变色相和饱和度，保留明度（最自然）
SOFTLIGHT 模式： 柔和增强，增加对比度
OVERLAY 模式：   强力增强，效果明显
HUE 模式：       只改变色相
SATURATION 模式： 只改变饱和度
```

### 3. 双曲线锐化法

超高清照片的专业锐化方法：

```
1. 复制图层两次
2. 第一个图层：
   - 高反差保留 (半径 3)
   - 混合模式：叠加
   - 不透明度：50%
3. 第二个图层：
   - 高反差保留 (半径 1)
   - 混合模式：亮光
   - 不透明度：30%
```

### 4. 局部对比度增强

```
1. 新建"曲线"调整图层
2. 拉出S型曲线（增加对比度）
3. 填充黑色蒙版
4. 用白色画笔涂抹需要增强对比的区域
```

## 📊 质量检查清单

修复完成后，检查以下项目：

- [ ] **清晰度**：细节是否清晰，无模糊感
- [ ] **色彩自然度**：肤色、天空、植被颜色是否真实
- [ ] **无明显噪点**：暗部和平坦区域是否干净
- [ ] **无损伤痕迹**：划痕、折痕是否已修复
- [ ] **对比度适中**：明暗对比是否平衡
- [ ] **分辨率足够**：是否达到预期的高清标准
- [ ] **整体和谐**：各元素是否协调统一
