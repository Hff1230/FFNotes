# 文档拍照去黑影 - 快速上手指南

## 🎯 30秒快速使用

```
1️⃣ 打开文档照片
2️⃣ 运行"文档去黑影.jsx"
3️⃣ 选择类型："试卷/单页文档"
4️⃣ 全部勾选选项（除了"黑白二值化"）
5️⃣ 强度选择："适中"
6️⃣ 点击"开始处理"
7️⃣ 保存为高质量JPEG
```

---

## 📸 拍照技巧（3分钟）

### 最佳设置

| 项目 | 设置 | 说明 |
|------|------|------|
| **光线** | 窗边自然光 | 避免直射，柔和均匀 |
| **角度** | 垂直俯拍 | 手机平行于文档 |
| **背景** | 深色布/桌面 | 与白纸形成对比 |
| **距离** | 30-50cm | 避免透视变形 |
| **分辨率** | 最高质量 | 至少2000px宽 |

### 拍照步骤

```
1. 找窗边，避免直射阳光
2. 铺深色布/纸在桌面
3. 放置文档，用重物压平
4. 手机开启网格线
5. 垂直俯拍，对焦文档
6. 保持稳定，点击拍摄
```

### 书本拍照

```
✅ 厚本书：
   - 用手机支架
   - 从45度角拍
   - 选择"书本（有装订阴影）"模式

✅ 薄本书：
   - 压平书页
   - 垂直拍摄
   - 选择"试卷/单页文档"模式
```

## 💡 专业技巧

### 1. 拍摄多张选最佳

```
同一文档拍2-3张：
- 不同光线角度
- 选择最清晰的一张处理
```

### 2. 分步处理

```
对于困难文档：
第1次：白平衡 + 去黑影
第2次：文字增强 + 锐化
第3次：去噪（如需要）
```

### 3. 使用历史快照

```
1. 窗口 → 历史记录
2. 处理前创建新快照
3. 随时恢复对比
```

### 4. 批处理

```
多张文档用相同设置：
1. 录制动作（窗口 → 动作）
2. 包含脚本运行
3. 文件 → 自动 → 批处理
```

## 🚨 注意事项

### ❌ 不要这样做

1. ❌ 使用闪光灯 → 造成强烈反光
2. ❌ 斜角拍摄 → 需要透视矫正
3. ❌ 过度锐化 → 文字出现伪影
4. ❌ 二值化后无法恢复 → 保留原文件
5. ❌ 低分辨率拍摄 → 打印模糊

### ✅ 应该这样做

1. ✅ 使用柔和均匀光线
2. ✅ 垂直俯拍，保持平行
3. ✅ 最高分辨率拍摄
4. ✅ 适度调整各图层
5. ✅ 保留原始照片备份

## 📱 手机拍摄设置建议

### iOS（iPhone）

```
设置：
- 启用"网格线"
- 关闭"HDR自动"（或手动选择）
- 开启"Live Photos"关闭（静态更好）

拍摄：
- 对焦：点击文档
- 曝光：下滑降低（避免过曝）
- 等待自动对焦完成
```

### Android

```
设置：
- 启用"参考线/网格"
- 分辨率：最高
- 格式：JPEG（兼容性）或RAW（质量）

拍摄：
- 专业模式（如有）：
  - ISO：100-200
  - 快门：1/60秒或更快
  - 对焦：单点对焦
```
