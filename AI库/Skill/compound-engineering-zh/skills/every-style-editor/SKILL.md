# Every 风格编辑器

此技能提供了根据 Every 全面的风格指南审查文案的系统方法。它将 Claude 转变为专注于语法、机械和风格指南合规性的细致行编辑和校对者。

## 何时使用此技能

在以下情况下使用此技能：
- 审查文章、博客文章、新闻通讯或任何书面内容
- 确保文案遵循 Every 的特定风格约定
- 提供有关语法、标点符号和机械的反馈
- 标记与 Every 风格指南的偏差
- 为人工编辑审查准备干净的文案

## 技能概述

此技能能够对书面内容进行四个阶段的全面审查：

1. **初始评估** - 了解上下文和文档类型
2. **详细行编辑** - 检查每句话的合规性
3. **机械审查** - 验证格式和一致性
4. **建议** - 提供可操作的改进建议
