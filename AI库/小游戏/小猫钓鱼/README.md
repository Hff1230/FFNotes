# 小猫钓鱼

经典的扑克牌接龙游戏，简单有趣，老少皆宜。

## 游戏规则

### 基本玩法

1. **发牌**：一副扑克牌（52张），平均分给两位玩家，每人26张，牌面朝下叠放
2. **出牌**：双方轮流从自己牌堆顶端翻出一张牌，放到中间的公共牌堆
3. **收牌**：当玩家出的牌与公共牌堆中某张牌的**点数相同**时，该玩家可以把两张相同点数牌之间的所有牌收走，放到自己牌堆底部
4. **胜负**：当一方手中的牌全部出完，游戏结束，另一方获胜

### 牌面点数

- A = 1点
- 2-10 = 对应点数
- J = 11点
- Q = 12点
- K = 13点

### 示例

```
公共牌堆：[3, 7, K, 5, 9, 3]
玩家出牌：3
结果：3与底部的3点数相同，收走全部6张牌！
```

## 操作说明

- 点击 **"出牌"** 按钮或按 **空格键** 出牌
- 点击 **"重新开始"** 开始新游戏

## 游戏技巧

1. 记住已经出过的牌，提高预测准确率
2. 观察对手剩余牌数，把握节奏
3. 牌多时要耐心，牌少时要果断
