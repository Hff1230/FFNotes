# 立体模型H5展示方案

## 📋 项目概述

本方案旨在通过交互式H5页面，直观展示三维立体物体与平面展开图之间的对应关系，帮助用户理解立体几何图形的展开原理和过程。

### 🎯 核心目标

- **可视化教学**：通过动画展示立体→平面的展开过程
- **交互学习**：用户可以手动控制展开进度，观察每个阶段
- **多角度展示**：3D立体视图 + 2D平面展开图同步展示
- **原理讲解**：详细说明每种立体图形的展开原理和应用

## 🎲 支持的立体图形

### 1. 正方体 (Cube)
```
展开特征：
- 面数：6个正方形面
- 展开图：十字形（4+1+1或3+1+1+1）
- 展开方式：沿任意棱剪开
```

### 2. 长方体 (Cuboid)
```
展开特征：
- 面数：6个长方形面
- 展开图：多种排列方式
- 展开方式：沿不同棱剪开有不同结果
```

### 3. 三棱柱 (Triangular Prism)
```
展开特征：
- 面数：5个面（2个三角形+3个长方形）
- 展开图：三角形-长方形组合
- 展开方式：沿侧棱剪开
```

### 4. 圆柱体 (Cylinder)
```
展开特征：
- 面数：3个面（2个圆形底面+1个曲面）
- 展开图：2个圆+1个长方形
- 展开方式：侧面沿高剪开成长方形
```

### 5. 圆锥体 (Cone)
```
展开特征：
- 面数：2个面（1个圆形底面+1个曲面）
- 展开图：1个圆+1个扇形
- 展开方式：沿母线剪开成扇形
```

## 🎬 展开过程展示方式

### 阶段一：立体状态（0%）
```
┌─────────┐
│         │
│   立体   │  ← 完整的三维物体
│  物体   │     所有面连接
│         │
└─────────┘
```

### 阶段二：部分展开（30%）
```
    ┌───┐
    │上 │
┌───┼───┼───┐
│左 │前 │右 │  ← 侧面开始展开
└───┼───┼───┘
    │下 │
    └───┘
```

### 阶段三：完全展开（100%）
```
    ┌───┐
    │上 │
┌───┬───┬───┬───┐
│后 │左 │前 │右 │  ← 完全平面化
└───┴───┴───┴───┘
    │下 │
    └───┘
```

## 🎨 交互功能设计

### 1. 图形选择
- 下拉菜单选择不同立体图形
- 切换时平滑过渡

### 2. 展开控制
- **自动播放**：一键自动展开
- **手动控制**：拖动进度条精确控制
- **暂停/继续**：随时暂停观察
- **重置**：快速回到初始状态

### 3. 视图控制
- **3D旋转**：鼠标拖拽旋转立体图形
- **缩放**：滚轮缩放查看细节
- **角度调整**：多角度观察

### 4. 信息展示
- **动态说明**：实时显示当前展开阶段
- **参数显示**：面数、棱数、顶点数
- **公式展示**：表面积、体积计算公式
