# 立体模型文件夹使用说明

## 📁 文件列表

1. **三维立体物体展开展示方案.md** - 完整的项目方案文档
2. **3d-unfolding-demo.html** - 主H5演示文件（纯CSS实现，推荐）
3. **3d-unfolding-threejs.html** - Three.js增强版（需要联网加载库）
4. **README.md** - 本说明文件

## 🚀 快速开始

### 方式一：直接打开（推荐）
1. 双击打开 `3d-unfolding-demo.html`
2. 在浏览器中即可直接使用，无需联网

### 方式二：本地服务器
```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js
npx serve
```
然后访问: `http://localhost:8000/3d-unfolding-demo.html`

## 📖 使用方法

1. **选择图形**: 从下拉菜单选择要展示的立体图形
2. **观看展开**: 点击"自动播放"观看完整展开动画
3. **手动控制**: 拖拽进度条手动控制展开程度
4. **旋转查看**: 在3D区域按住鼠标拖拽可旋转查看物体

## 🔧 自定义扩展

### 修改颜色
在HTML文件中搜索 `rgba()` 修改各面颜色：

```css
.cube-face.front {
    background: rgba(255, 107, 107, 0.8); /* 前面颜色 */
}
```

### 添加新图形
1. 在 `shapeData` 对象中添加新图形数据
2. 创建对应的HTML结构
3. 添加CSS样式

### 调整动画速度
修改JavaScript中的进度增加量：

```javascript
progress += 0.5; // 调整这个值改变速度
```

## 🐛 常见问题

**Q: 3D物体不旋转？**
A: 检查浏览器是否支持CSS 3D transforms

**Q: 动画卡顿？**
A: 尝试降低旋转速度或使用更快的设备

**Q: Three.js版本无法使用？**
A: 确保已连接互联网，Three.js需要从CDN加载
